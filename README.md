# Contesturl

Web上で開催されているプログラミングコンテスト(e.g. [AtCoder](http://atcoder.jp)の問題URLを取得できます。

## Installation

```ruby
gem 'contesturl'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install contesturl
